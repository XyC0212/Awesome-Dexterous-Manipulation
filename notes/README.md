# Notes

One file per paper (named after the paper's slug) for **your own opinions and longer notes**.
The `## My take` section is searched during retrieval, so jot down why a paper matters to you.
Objective metadata stays in the [README table](../README.md) and [SUMMARIES.md](../SUMMARIES.md).

A stub is created automatically when a paper is added — just fill in the sections.
